# Part 5 — AI Race Engineer Chatbot (Live 2024/2025 Data)

An AI-powered race engineer chatbot that answers natural
language questions about F1 races using live FastF1 data
from the 2024 and 2025 seasons.

## What it does

Ask it anything about recent races:

- "Who had the fastest lap at Monaco 2024?"
- "Compare Verstappen and Norris tyre strategy at Silverstone 2024"
- "Which driver improved the most from qualifying to race in 2024?"
- "What was McLaren's average pit stop time in 2024?"

It responds with data pulled live from FastF1 + a plain
English explanation generated by an LLM.

## Why this is impressive

Traditional dashboards show fixed charts.
This chatbot lets you ask any question in plain English
and get a data-backed answer — like having a real
F1 data engineer on demand.

## How it works

1. You type a question in natural language
2. LLM extracts the race, driver, metric from your question
3. FastF1 fetches the relevant live 2024/2025 race data
4. LLM formats a clear answer with numbers and explanation

## Tech stack

- Python 3.11
- FastF1 — live 2024/2025 F1 telemetry
- LLM API (Claude / OpenAI)
- RAG (Retrieval Augmented Generation)
